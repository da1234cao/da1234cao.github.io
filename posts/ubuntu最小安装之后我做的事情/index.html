    安装zsh

    sudo apt install zsh

    下载oh my zsh

    git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh

    创建配置文件

    cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

    设置zsh为默认的shell

    chsh -s /bin/zsh

    查看是否设置成功，我是重启之后才看到效果，不重启能生效的方法我目前不知道

    echo $SHELL

主题，我使用的是默认主题

ZSH_THEME=”robbyrussell”

插件：跳转，提示，语法高亮，git

plugins=(zsh-autosuggestions zsh-syntax-highlighting z git) git clone git://github.com/zsh-users/zsh-autosuggestions $ZSH_CUSTOM/plugins/zsh-autosuggestions git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
